
"How it works?" page:

Entry point
* Link/button in hero section, visible to all users

Navigation
* Page view (same as Practiced Words/Groups/Chars)
* Sticky header + X close

Content
* What is this — vocab learning app, interactive practice + printable workbooks
* Groups — one group = one study session, stable IDs for progress tracking
* Practice — stroke writing, pinyin typing, smart word ordering, auto-pick next group via Practice button
* Workbook — print, accordion fold, practice by hand, unfold to self-check
* Progress — per-word/char stats, session history, progress bars, activity calendar, cross-device sync

Tone
* Short, friendly, practical
* No walls of text

Out of scope
* Video tutorials, interactive onboarding, per-kind variations
